Purpose

This driver provides a set of functions to manage:

  • LEDs
  • Push-buttons
  • COM port
  • Sensors (accelerometer, gyroscope, temperature)
  • EEPROM

available on B-WBA6M-WPAN Kit from STMicroelectronics
